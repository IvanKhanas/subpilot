# SubPilot

SubPilot is a Kotlin/Spring Boot microservice backend for a Telegram AI chat
with usage limits, paid subscriptions, and payment processing.

This repository is the starting point for developing the platform with
these services:

- `tg-bot-service`
- `chat-service`
- `subscription-service`
- `payment-service`
